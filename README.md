# ABDM API documentation

Public documentation for the ABDM API, maintained by Unimetacare.

## Contents

- **Getting started** — Overview and integration context (`index.mdx`)
- **API reference** — OpenAPI-backed endpoint docs under `api-reference/`

Configuration for the documentation site is in `docs.json`. API definitions live in `api-reference/openapi.json`.

## Local preview

If your team uses the standard docs CLI for this repository:

1. Install the CLI globally: `npm i -g mint`
2. From the repository root, run: `mint dev`
3. Open the preview URL shown in the terminal (often `http://localhost:3000`)
